How to play

            * New Game
            1. Enter the game code. Game code can be anything for e.g. 1234, test, play etc
            2. Click Start Game : Start game will create a new game and you will play as white.
            3. Join game as white assist : You will join the game as 2nd player on white team.
            Note the game has to be created before by player1 aka white
            4. Join game as black : You will join the game as black team. Game has to be created first.

            
            * Continue Game
            1. Enter the game code and username
            2. Click Continue Game
            3. Game will be continued
